Mining Instructions

To start mining with Diamond Drillers BCH, you need to follow the steps below:

  1. Download and install a Bitcoin Cash full node on your device. You can download the official Bitcoin ABC client from the Bitcoin ABC website.
  2. Once you have installed the Bitcoin ABC client, open the configuration file and add the following line:

    stratum=+tcp://diamonddrillersBCH:8333

    This will display the stratum as "diamonddrillersBCH" while still connecting to the original stratum at "192.168.0.58:8333".

  3. Save the configuration file and restart the Bitcoin ABC client.
  4. Connect your mining software to the stratum "diamonddrillersBCH:8333". You can use any mining software that supports the SHA-256 algorithm, such as CGMiner or BFGMiner.
  5. Start mining! Your mining software will automatically connect to the Diamond Drillers BCH mining pool and start mining Bitcoin Cash.
